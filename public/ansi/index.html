Il existe un moyen de personnaliser la couleur de vos chaines de caractères 🥝

ANSI escape code

Un standart souvent compris dans les terminaux

print("\033[1mHello\033[0m, \033[0;36mWorld!\033[0m") 
       └Souligné     │          │
                     └Remise à 0│
                                └Cyan

Résultat:

Hello, World!
